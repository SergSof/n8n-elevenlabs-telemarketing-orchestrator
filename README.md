# n8n-elevenlabs-telemarketing-orchestrator
n8n workflow template for automated outbound AI telemarketing calls with CRM queue processing, ElevenLabs voice calls, callback handling, and CRM status updates. Шаблон n8n workflow для автоматизации исходящих AI-звонков: обработка очереди CRM, блокировка записей, запуск звонков через ElevenLabs, обработка callback и запись финальных статусов в CRM
